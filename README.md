# TBC Detection System - Frontend

Frontend application for Tuberculosis Detection System.

## 🚀 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router

## 📁 Project Structure

src/
 ├── components/
 ├── pages/
 ├── services/
 ├── hooks/
 ├── layouts/
 └── utils/

Clone the repository:

```bash
git clone https://github.com/Software-Deteksi-sel-TBC/TBC-Frontend-web.git
cd TBC-Frontend-web
